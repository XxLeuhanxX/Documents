### Descripción del Proyecto

Este proyecto está enfocado en la creación de un ransomware de prueba con fines educativos y de concientización sobre la importancia de la seguridad cibernética. El ransomware utiliza el módulo `cryptography` y la librería `Fernet` para encriptar y desencriptar archivos de un directorio especificado.
El fichero file contiene un gif , una imagen y un archivo de texto , al ejecutar encrypt.py se encriptan los archivos del fichero y se crea una key para su desencriptacion ,por las moscas no eliminar el archivo key.key si no a desencriptado los archivos del fichero files. 
finalmente con decrypt.py se desencripta.

### Instrucciones de Uso

1. **Instalación de Dependencias**
   - Asegúrate de tener instaladas las librerías `cryptography` y `Fernet` en tu entorno de Python.

2. **Ejecución del Ransomware**
   - Utiliza los archivos ejecutables `run_encrypt.bat` y `run_decrypt.bat` para ejecutar los scripts de encriptación y desencriptación respectivamente.

3. **Cambio de Clave (Opcional)**
   - Si deseas cambiar la clave de encriptación, elmina el archivo key.key y ejecuta en tu entorno el archivo encrypt.py

### Aviso Legal

Es importante recordar que este proyecto se realiza con fines educativos y de concientización sobre la seguridad cibernética. No se debe utilizar con fines maliciosos o ilegales. Se recomienda utilizarlo en entornos controlados y bajo consentimiento de los propietarios de los archivos.
